## Todolist

### Стек

react_vite, node, express, sqlite3, jwt, tailwind

### Что сделано

В приложении реализована регистрация и авторизация пользователя, а также сам Todolist.
В todolist добавлять задачу, делать пометку, что задача выполнена и удалять задачу.


### Установка

1. Клонировать репозиторий:

```bash
git clone https://github.com/Natalia-Vorobeva/react_vite_node_express_sqlite3_jwt_tailwind.git

cd react_vite_node_express_sqlite3_jwt_tailwind
```

2. Установить зависимости для client и server

```bash
npm install
cd server && npm install && cd ..
cd client && npm install && cd ..
```

## Запуск

Приложение запускается из корневой папки:

```bash
npm run start
```

Это приведет к одновременному запуску сервера разработки Vite и экспресс-сервера.
